    vertical-rhythm

    @mixin vertical-rhythm($font-size, $bottom-rows: 1, $top-rows: 0, $use-height: false, $border: 0) { ... }

    Description

    Maintain the Vertical-Rhythm

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $font-size

    A value in px or rems

    		Value— none
    $bottom-rows

    Allow the specification of additional base-line rows below the element

    		String1
    $top-rows

    Allow the specification of additional base-line rows above the element

    		String0
    $use-height

    If true it uses height for height-based element (block-level elements or replaced inline elements), otherwise it uses line-height

    		Booleanfalse
    $border

    On non-replaced inline elements (span, paragraphs) it solves the problem with borders and margin collapsing

    		Value0

    Example

    Usage

    .namespace_myClass {
   @include vertical-rhythm(2rem);
 }

    CSS output

    .namespace_myClass {
  height: 2rem;
  margin-top: 0.5rem;
  margin-bottom: 2rem;
}


.namespace_myClass {
   @include vertical-rhythm(2rem, $use-height: true);
 }

    CSS output

    .namespace_myClass {
  font-size:      2rem;
  line-height:    3rem;
  margin-top:     0;
  margin-bottom:  1.5rem;
}


.namespace_myClass {
   @include vertical-rhythm(350px, $use-height: true);
 }

    CSS output

    .namespace_myClass {
  height: 21.875rem;
  margin-top: 0.3125rem;
  margin-bottom: 1.8125rem;
}

    Requires

    • [function] to-string
    • [function] rem
