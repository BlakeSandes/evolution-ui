Spiral Text Affect

Often on the hover state of a link or button, or even a small body of text, an effect is given to bring attention to the action. The spiralling of text is such an effect that could add a little flare to the hover state.

The affect can be placed on either a <button> or <a> element as well as on any text surrounded by <span> tags. The element gets the evo_c-spiralText class for the affect to take place and each individual text is required to be wrapped by an <em tag.

Buttons and links wil need the __button class appended to have the proper appearance like so: evo_c-spiralText__button.

Background Size and Shape

The text background on buttons and links can be changed both in size and shape.

The background size by default is styled with adequate padding and the shape is rectangular. Yet, can also be displayed full height and with a circular shape.

The full height background is displayed with the appended classes of --fullHeight or circlefullHeight. If the circle background is desired but not full height, simply append the class of --circle instead of circlefullHeight.
