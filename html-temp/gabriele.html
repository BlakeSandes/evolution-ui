Progress Bars

Add the class .su_progress-bar to implement a progress bar element. Then, add the .su_bar class to the child span element. Examples:

To implement the progression add a value between 0% and 100% to the width property of a bar element.

Add one of the following classes to a bar to get some extra colors:

  • Add the class .su_bar--success to get a light green bar
  • Add the class .su_bar--warn to get a light pink bar
  • Add the class .su_bar--error to get a dark pink bar

Add the class .su_progress-bar--rounded to a progress-bar element to make its borders rounded. Examples:

Infinite Loading

Add the class .su_progress-bar--indeterminate to a progress-bar element to make it an indeterminated progress bar. Examples:

Backward

Add the class .su_bar--b to the child span element of the progress-bar element to get a backward animation.

Forward and Backward

Add the class .su_bar--fb to the child span element of the progress-bar element to get a forward and backward animation.

Forms

Add the class .su_form to a form element. Then add each input element inside a container field and give it the class .su_field.

Add the class .su_field-label-up to the field to get the label up when the user click inside the input.

Add the class .su_input to every input element.

Add the class .su_has-error to the field and add an element with class .su_error-message if you want to provide an error message to the end-user.

Add the class .su_input-checkbox to a seed checkbox element.

Add one of the following classes to a checkbox or radio to get some extra colors:

  • Add the class .su_input-checkbox--red to get a red checkbox
  • Add the class .su_input-checkbox--green-dark to get a dark green checkbox
  • Add the class .su_input-checkbox--pink to get a pink checkbox
  • Add the class .su_input-radio--red to get a red radio
  • Add the class .su_input-radio--green-dark to get a dark green radio
  • Add the class .su_input-radio--pink to get a pink radio

Add the class .su_input-checkbox--x to a checkbox element.

Add the class .su_input-radio to a radio element.

Add the class .su_input to button element or add the class both the classes .su_input .su_input--button to a <A> or <DIV> element.
Add the class .su_input--button--3D to get a 3D button.

Add one of the following classes to a button to get some extra colors:

  • Add the class .su_input--button--dark-pink to get a dark pink button
  • Add the class .su_input--button--light-green to get a light green button
  • Add the class .su_input--button--green to get a green button

Forms Boxes

Add the class .su_form-box to a container and use the class .su_form-box--border if you want to give it a rounded border.

Add the class .su_field--border to get a field with a rounded edge.

Add the class .su_input--full-width to make the input full width (100%)

Add the class .su_field--border to get a field with a rounded edge.

Add the class .su_input-icon icons' containers and use the .su_input-icon--up to get the icon up to the input.
