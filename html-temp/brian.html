Float Classes

Add the class .su_left or .su_right to float an element left or right respectively. Then, add the class .su_clear class to clear floated elements. Examples:

Full Width Classes

The full width classes expand the element they are applied to to the full width of it's parent element using the .su_full-width class. Examples:

Radius & Round Classes

Add the class .su_radius to apply the default radius size on elements. Add the class .su_round class to create a pill or capsule rounding effect on elements. They can be applied to many different elements. Examples:

Color Classes

Seed UI Framework comes with a number of built in colors that can be customized for your website's style guide. You can change the background color of elements or the text color by applying their associated classes. Examples:
