Float Classes

Use the .su_lowercase class to transform text of an element to all lowercase.

use the .su_uppercase class to transform text of an element to all uppercase.

Use the .su_capitalize class to transform text of an element to use uppercase letters for the beginning of each word.

Spacer Classes

A collection of classes that have varying margin values on different sides for quick spacing fixes.

.su_space-top--small applies a margin-top: 20px

.su_space-top--medium applies a margin-top: 50px

.su_space-top--large applies a margin-top: 100px

.su_space-right--small applies a margin-right: 20px

.su_space-right--medium applies a margin-right: 50px

.su_space-right--large applies a margin-right: 100px

.su_space-bottom--small applies a margin-bottom: 20px

.su_space-bottom--medium applies a margin-bottom: 50px

.su_space-bottom--large applies a margin-bottom: 100px

.su_space-left--small applies a margin-left: 20px

.su_space-left--medium applies a margin-left: 50px

.su_space-left--large applies a margin-left: 100px
