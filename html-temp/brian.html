Float Classes

Add the class .su_left or .su_right to float an element left or right respectively. Then, add the .su_clear class to clear floated elements. Examples:

menuDiv 2 Floated Left

Color Classes

Seed UI Framework comes with a number of built in colors that can be customized to your website's style guide. You can change the background color of elements or the text color by applying their associated classes. Examples:

Visibility & Hidden Classes

Add the class .su_hide to an element to remove it from the page view. The number 4 span element in the following list is hidden:

1 2 3 4 5 6

Add the class .su_invisible to make the visibility of an element hidden but still retain its space in the DOM. The number 4 span element in the following list is invisible:

Text Transform Classes

Use the .su_lowercase class to transform text of an element to all lowercase.

use the .su_uppercase class to transform text of an element to all uppercase.

Use the .su_capitalize class to transform text of an element to use uppercase letters for the beginning of each word.
